# Reto_PowerQuery_Festivos_2023
Reto de Power BI y Power Query para la creación de una tabla calendario de 2023 y su integración con los festivos de España obtenidos desde una página web. Se utilizan funciones de Power Query para la extracción, transformación y combinación de datos, así como el modelado mediante relaciones entre tablas.


## Objetivo

Crear una tabla calendario para el año 2023 y enriquecerla con los festivos de España obtenidos desde una página web utilizando Power Query.

## Tecnologías utilizadas

- Power BI
- Power Query (M)
- Web.Contents
- Web.Page
- Table.AddColumn
- Table.NestedJoin
- Table.ExpandTableColumn

## Resultado

El proyecto genera una tabla calendario con:

- Fecha
- Día de la semana
- Mes
- Número de mes
- Día del mes
- Festivo (si aplica)

Además, se establece una relación entre la tabla Calendario y la tabla Festivos.
